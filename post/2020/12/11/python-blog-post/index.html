Python blog post

December 11, 2020   

Most of my Python skills were developed through various other classes I have taken through the elements of computing certificate. Some of these classes include data structures, software engineering, databases, and web programming. However I did pickup a few new skills from this class, including exposure to pandas and other Python libraries, and learning how to have R and python talk to each other with reticulate.

I learned that to look at datasets in Python, you must import pandas, a software library that is used for data manipulation and analysis. Pandas offers data structures and operations that can be used for reading and manipulating data. For example: {python} import pandas as pd quakes=pd.read_csv(“https://vincentarelbundock.github.io/Rdatasets/csv/datasets/quakes.csv",index_col=0) quakes.shape

Another interesting function from Pandas is the describe function, which returns the number of variables along with some properties of the dataset, such as mean, standard deviation, min, max, and quartiles. For example: {python} import numpy as np mag=quakes.mag mag.describe() produces count 1000.000000 mean 4.620400 std 0.402773 min 4.000000 25% 4.300000 50% 4.600000 75% 4.900000 max 6.400000 Name: mag, dtype: float64

One library I learned how to use is NumPy. NumPy stands for Numerical Python and is a Python library that is used for working with arrays and contains functions such as min, max, mean and median. Some examples include: {python} np.median(quakes[quakes.long>175].mag) np.mean(np.median(quakes[quakes.depth>247].mag)) np.std(quakes[quakes.depth>247].lat,ddof=1)

Another Python library I made use of is Matplotlib, a library for creating plots and visualizations in Python. Some functions in matplotlib include boxplot(), hist(), and scatter()

I also learned how to have R and Python talk to each other. The reticulate package allows R and Python to talk to each other. For eample, I can use Pandas to read and manipulate data, and then use ggplot2 from R to plot the data. Some example code: {python} side, main = plt.subplots() fiji = plt.imread(“http://valorielord.com/default/cache/file/D4256462-E44C-32F4-62C9A3C4AA762918_bodyimage.png") main.imshow(fiji, extent=[135,215,-40,0]) plt1 = main.scatter(x= quakes.long ,y= quakes.lat, alpha=.2, c= depth) side.colorbar(plt1)

plt.show()

Regular expressions in Python were new to me. Regular expressions are used to match and return parts of real world strings. For example, using re.findall(), I can match all numbers, IP addresses or MAC addresses and pull them from a string. Some example code: {python} temp = re.findall(r'\d+', string1) ip = re.findall(r'[0-9]+(?:.[0-9]+){3}', string2) found = re.findall(r'(?:[0-9a-fA-F]:?){12}',string2)

Overall I learned a repectable amount about Python, despite my previous knowledge, especially about having R and Python talk to each other with reticulate and about Python libraries involving data manipulation.
